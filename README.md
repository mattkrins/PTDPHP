## PTDPHP (Podio-to-Disco PHP)

![Exporting](http://i.imgur.com/Ais9oPy.png)

#### About:
This is a php based exporter script to convert [Podio](https://podio.com/) assets to [Disco](http://discoict.com.au/) devices.

#### Prerequisites:
* PHP 5
